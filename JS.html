變數類型
var型別可隨意賦值，變數名稱可重複，有全域變數的特性：

var var1 = value1

let型別必須設定初始值，變數名稱不可重複，為普通變數：

let var2 = value2

const型別必須設定初始值，不可修改：

const var3 = value3
