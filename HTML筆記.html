CSS程式碼

CSS基礎語法
  • 段落：＜div＞
    • 在＜div＞後加上style="某項屬性 : 某項的設定值 ; 另一項屬性:另一項設定值"
    因為＜div＞標籤會使文字強制換行，若只想在行內加上屬性，可以使用＜span＞標籤
    • 字體：font-weight ex. font-weight:bold(粗體)
    • 文字大小：font-size ex. font-weight:20px
    • 顏色：color ex. color:green
  • CSS Box Model
    • 與某方向區塊的距離：margin
      • ex. margin-top:10px ; margin-bottom:10px
    • 邊框：border
      • ex. style="border:(寬度)2px (樣式)solid (顏色)blue
    • 填充：padding
      • ex. style="padding:5px ; border:1px dashed red
    • 背景顏色：background-color
      • ex. width:500px;background-color:pink
    • 寬度：width
      高度：height *只能在區塊式的標籤內使用這兩種屬性
      • ex. style="width:800px;height:40px;border:1px solid orange"
    • 定位：position
      絕對定位：absolute 以網頁最左上角為定位基準點進行定位
      相對定位：relative 以程式碼所在區塊的左上角為基準點進行定位
  • CSS選擇器
    • 在＜head＞區塊使用＜style＞標籤，在＜style＞標籤加入屬性type="text/css"
      在＜style＞標籤內以{}包裹想使用的屬性設定，在{前方加入對此設定的命名並在名稱的前方加上dot(.)
      在使用上以屬性 class=設定名稱 來做對應
      ex.
      .keyword{color:red;font-weight:bold} --> ＜span class="keyword"＞
      .border{border:2px solid blue} --> ＜span class="border"＞
